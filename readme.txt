Adds a Edd Pending Notification  ADD-on. Requires Easy Digital Downloads Plugin.

== Description ==

Add a Send email quick action to all pending payments to easily remember the user about his pending order.


== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

= Using the Plugin Manager =

1. Click Plugins
2. Click Add New
3. Search for `Edd Pending Notification`
4. Click Install
5. Click Install Now
6. Click Activate Plugin
7. Now you can see in the wordpress menu the new option templates of pending emails to create your email template and then in payment history you can see a new button to 'send mail' in any payment that is pending

= Manually =

1. Upload `edd-pending-notification` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
